IOs (input/output)

Digital

  • GND = 0V = LOW
  • VCC = 5V = HIGH

Digital


// write to pin 11
digitalWrite(11, HIGH);
digitalWrite(11, LOW);

// read from pin 12
digitalRead(12) == HIGH;
digitalRead(12) == LOW;

Analog

  • 2.5V => 0V------|------5V
  • 1.0V => 0V--|----------5V
  • 3.0V => 0V--------|----5V
  • ...


  • only read
  • 10bit resolution = 1024 possibilities
  • 5v / 1024 = 0.004V = 4mV

Program structure

Setup

  • Init IOs direction
  • Init Serial
  • Init communication
  • Set initial state

Setup


void setup()
{
    pinMode(11, OUTPUT);
    pinMode(A0, INPUT);
    Serial.begin(9600);
    digitalWrite(11, HIGH);
}

Loop

  • Loop fast !
  • Logic of your program here
  • can be paused

Loop


void loop()
{
    digitalWrite(12, HIGH);
    delay(1000); // Wait for 1000 millisecond(s)
    digitalWrite(12, LOW);
    delay(500); // Wait for 500 millisecond(s)
}
