Angular 2

  • Developed in Typescript
  • Components, no more directives
    and controllers
  • no more $scope
  • Pipes, no more filter
  • New dependency injection
  • New router
  • RxJs (Obeservable and cie)

Material design

  • Specifications and guide line
  • Developed by google (2014)
  • Mobile oriented
  • Rapidly used in Android and new
    google web application (Maps, Inbox, etc.)

Solutions tested

    - library
  • Angular2 and polymer
    - framework

Angular 2 material

  • Pros
    • Library by google
    • Very good integration
      with Angular 2
  • Cons
    • In Alpha
    • Too few components
https://material.angular.io

Polymer

  • Pros
    • Lots of components
    • Developed by Google
    • Work with Angular 2
  • Cons
    • Half suported by Firefox
    • Don't even ask about IE...
https://www.polymer-project.org

Bootstrap Material design

  • Pros
    • Use Boostrap elements
    • Nothing new to learn
  • Cons
    • Conflict with liferay CSS
    • We must trigger JS
      functions from angular
https://github.com/FezVrasta/bootstrap-material-design

Final solution

  • Angular 2
  • Material design
  • Gulp as build system
  • Standalone development
  • Iframe in liferay
